# ADR-0002 — Українська мова за замовчуванням

## Статус
Прийнято.

## Рішення
Усі інтерфейси, документи, звіти та пояснення FHOS створюються українською мовою.

## Винятки
Міжнародні медичні терміни, стандарти, API, назви бібліотек і медичні скорочення можуть залишатися в оригіналі, але мають отримувати українське пояснення.
