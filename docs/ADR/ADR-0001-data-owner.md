# ADR-0001 — Дані належать користувачу

## Статус
Прийнято.

## Рішення
Усі медичні дані у FHOS належать користувачу або члену сім'ї, до якого вони відносяться.

## Наслідки
- Експорт даних є обов'язковою функцією.
- Система не передає дані третім сторонам без явної згоди.
- Оригінальні документи не змінюються.
