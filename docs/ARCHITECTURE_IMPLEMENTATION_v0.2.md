# FHOS Architecture Implementation v0.2

## Що реалізовано в коді

### API Core
Єдина точка підключення модулів. Відповідає за маршрутизацію запитів між модулями.

### AI Core
Перший сервісний шар для майбутніх AI-агентів. Зараз повертає статус і принцип роботи.

### Medical Core
Перший сервісний шар для формування медичного профілю. Поки працює без реальних даних.

### Laboratory Engine
Перший сервісний шар для майбутнього імпорту та аналізу лабораторних показників.

## Поточна мета

Не додавати складність, а зробити перевірену основу, яка запускається і має чіткі межі модулів.
